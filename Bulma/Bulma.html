Duurzaamheid

3/5

Voor bulma qua duurzaamheid is het wat apart het heeft een iets mindere userbase op github(240k). Maar krijgt voldoende updates en is nog in volle groei daarmee dit cijfer.

Lage leercurve

3.5/5

Bulma is makkelijk te begrijpen de classes zijn iets langer maar lijkt erg op het gebruik van een taal zoals de woorden is en has.

Leesbare code

4/5

De code makkelijk te lezen alleen kan het wat lang zijn omwille van hoe bepaalde properties noemen.

Grote community

3/5

Bulma is de 2e grootste community van de 4 frameworks die we hebben bekeken. Het heeft een userbase van 240k op github en is nog steeds in volle groei.

Onderhoud

4/5

Op het vlak van onderhoud heb ik een 4 gegeven omdat het op lange termijn genoeg updates en ondersteuning biedt.

Responsiviteit

4/5

Bulma is een responsive framework dat gebruik maakt van flexbox en grid. Dit maakt het makkelijk om een responsive website te maken.

Animaties

2.5/5

Bulma heeft niet veel ingebouwde animaties maar kan wel via Sass maar is niet altijd handig.

Customizable

4/5

Bulma heeft de mogelijkheid om custom css te schrijven maar niet rechtstreeks zoals bij Tailwind. Dit kan door middel van Sass wat uitgebreid uitgelegd is in de documentatie.

Genoeg documentatie

4/5

De documentatie is voldoende aanwezig ook om custom css te schrijven of toe te voegen.
