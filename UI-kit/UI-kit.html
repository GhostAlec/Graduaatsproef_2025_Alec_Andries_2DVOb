Duurzaamheid

Lage leercurve

4/5

UI kit heeft een lage leercurve er zijn veel voorgebouwde componenten aanwezig die gebruikt kunnen worden en is makkelijk de classes te onthouden.

Leesbare code

4.5/5

Heeft een makkelijke naam geving voor zijn classes maar kunnen soms wel wat lang zijn.

Grote community

2.5/5

UI kit de community is niet heel groot waardoor het qua populariteit wel minder is tov andere frmaeworks.

Onderhoud

3.5/5

UI kit krijgt genoeg en vooral op regelmatige basis updates waardoor het op lange termijn kan ondersteunen.

Responsiviteit

4/5

UI kit heeft een goede ondersteuning voor mobile devices je kan kiezen tussen het gebruik van grid of flexbox en de ingebouwde componenten zijn ook mobile friendly.

Animaties

3/5

Op het vlak van animaties is UI kit niet zo sterk maar het heeft wel een paar leuke animaties die je kan gebruiken die ook veel voorkomen zoals hovers,... .

Customizable

3/5

Voor custom css geef ik een 3 omdat het niet zo gemakkelijk is om moest het nodig zijn aanpassingen te doen kan wel door de classes te overwriten maar kan snel complex worden.

Genoeg documentatie

4.5/5

Er is voldoende documentatie aanwezig ook voor hoe je classes overwrite of aanpast.
